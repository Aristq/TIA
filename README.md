85. Что такое транспонирование массивов в библиотеке NumPy? Опишите правила
Транспонирование — изменение формы массива, при котором строки становятся столбцами, а столбцы — строками. Правила:
Для 1D массива форма не меняется
Для 2D массива строки и столбцы меняются местами
Для многомерных массивов изменяется порядок осей

96. Как можно презентовать результаты мониторинга информационного пространства? В каких случаях используются клиппинг, дайджест, сводка и другие форматы?
Форматы презентации:
Сводка — краткий обзор ключевых событий
Дайджест — структурированный анализ по темам
Клиппинг —  это подборка и анализ публикаций в СМИ, социальных сетях и блогах за определенный период времени, отражающая упоминания объекта мониторинга (например, бренда, персоны или события). . При презентации результатов клиппинг является отчетом, который визуализирует эти материалы, часто включая выжимки, ссылки, статистику охвата и оценки эффективности
Отчет — детальный анализ с выводами

Применение:
Сводка — для ежедневного информирования
Дайджест — для еженедельных отчетов
Клиппинг —  это подборка и анализ публикаций в СМИ, социальных сетях и блогах за определенный период времени, отражающая упоминания объекта мониторинга (например, бренда, персоны или события)
Отчет — для глубокого анализа
